## 프로젝트 소개

<p align="justify">
감정소비(스트레스 해소를 위해 소비하는 것)를 경험한 사람들이 많다는 기사를 보고 감정소비를 위한 사이트를 만들어보고자 했습니다.
이용자 간에 감정을 공유할 수 있는 커뮤니티를 만들어 커뮤니티 커머스로 제작해보았습니다.
</p>

<br>
